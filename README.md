# leyley bot

<img src="./media/imagens/icon.jpg">

leyley bot e uma botzinha de telegram com diversos recursos e comandos pensada para o seu grupo :3

## INSTALAÇÃO NO TERMUX
### primeiramente, baixe algumas das  dependencias básicas caso ainda não tenha

```bash
pkg update && pkg upgrade
```
```bash
pkg install git python -y
```

### em seguida, baixe o codigo fonte e as bibliotecas necessárias
```bash
git clone <futuro_link> && cd leyley_bot
```
```bash
pip install -r requirements.txt
```

## configuração do bot 
entre no telegram e pesquise pelo nomd "@BotFather" para iniciar a conversa

em seguida use o /newbot e escolha um nome para o seu bot 

copie e cole o token no arquivo config.json
